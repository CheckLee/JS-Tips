# JS - 二分查找
> [算法指北]() - 有事没事两个指针；数组按照大小排序过的

## 非递归版本

1. 头尾两个指针
2. 如果`target`小于中间值，那么就在数组前部分查找
3. 如果`target`大于中间值，那么就在数组后部分查找
4. 如果没找到，就重复第一步直到找到或者 **头指针到尾指针后面**

## 递归版本思路

> 递归版本和非递归版本思路一致

等价于求解`f(n)`函数目标，分解可能为：

* `f(0, n/2)`函数目标
* `f(n/2, n)`函数目标

```JavaScript
f(n) = 
{
  f(0, n/2) // 进入该步骤条件
  f(n/2, n) // 进入该步骤条件
}
```

对于`f(0, n/2) or f(n/2, n)`重复上诉步骤步骤，这是递归的过程。
